In questa directory salvare esclusivamente i "deliverable" relativi alla consegna 
per le valutazioni. In linea di massima i deliverable saranno due.

Il primo viene redatto durante la frequenza al corso e fornisce  le descrizioni di 
massima relative al proprio progetto.

Il secondo viene redatto alla consegna finale e contiene tutta la documentazione 
di progetto che si ritiene utile ai fini della valutazione.

ENTRAMBI I DELIVERABLE DEVONO ESSERE PRESENTI SOTTO FORMA DI FILE UNICO (*.pdf o *.doc). 
IN PARTICOLARE E' VIETATO SALVARE IMMAGINI, SCREENSHOT etc... PER RENDERLE DISPONIBILI
AL DI FUORI DEL FILE CHE RAPPRESENTA IL DELIVERABLE.
